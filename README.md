# Java File Operations – Notes App
This is a beginner-friendly Java project to practice **file handling basics**.

## Features
This project performs all basic file operations:
1. Create file  
2. Write data  
3. Read data  
4. Append data  
5. Rename file  
6. Copy file  
7. Delete file  

## Example Used
Notes App – stores simple text notes.
Each line is one note:
Today I learned Java file handling.  
File operations are easy.  
Practice daily.

## Technologies
- Java
- IntelliJ IDEA

## Files Used
- notes.txt → initial file  
- my_notes.txt → renamed file  
- backup_notes.txt → copied backup  

## How to Run
1. Open project in IntelliJ  
2. Run `NotesApp.java`  
3. Check project folder for created files  

## Learning Outcome
After this project, you will understand:
- What is a file and directory  
- How Java creates and manages files  
- How to read, write, append data  
- How to rename, copy, and delete files  

This project is best for beginners to understand Java file handling clearly.
